# Cerememory TypeScript SDK

TypeScript/Node.js bindings for Cerememory, built with [napi-rs](https://napi.rs/).

**Status**: Not yet implemented. See [Roadmap](../../README.md#roadmap).

## Planned Features

- Full CMP protocol access from TypeScript
- Custom LLM adapter development in TypeScript
- Integration with LLM SDKs (Anthropic, OpenAI, etc.)
