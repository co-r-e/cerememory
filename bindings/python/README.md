# Cerememory Python SDK

Python bindings for Cerememory, built with [PyO3](https://pyo3.rs/).

**Status**: Not yet implemented. See [Roadmap](../../README.md#roadmap).

## Planned Features

- Full CMP protocol access from Python
- Custom LLM adapter development in Python
- Integration with the broader AI/ML ecosystem (LangChain, LlamaIndex, etc.)
